# Camada Cross Cutting

Contém código que é transversal a várias camadas

Pode conter interfaces ou implementações de serviços que são usads em toda a aplicação.

É uma camada que facilita a comunicação e integração entre camadas.

Será usada para :
- Configuração de serviços
- Injeção de depêndencia
- Registro dos serviços do contexto
- Registro do serviço do repositório
